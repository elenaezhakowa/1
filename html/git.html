Git

  • git init - Инициализация репозитория
  • git clone ссылка с GitHub - Клонировать удаленный репозиторий с GitHub себе на компьютер
  • git status - Просмотр изменений
  • git add "имя файла" - добавить файл в отслеживаемые
  • git commit -m "Наименование" - сделать коммит
  • git push - Внести изменения на удаленный репозиторий
  • git log - Проверить изменения
  • git pull - внести изменения из удаленного репозитория в локальный
